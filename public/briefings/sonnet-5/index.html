Claude Sonnet 5 · for people who already build with Claude

The mid-tier model drew even with the flagship.

Claude Sonnet 5 is Anthropic’s mid-tier model — and on the work most teams run all day, it now performs close to Opus 4.8, at a lower price.

You already know how to drive Claude. This is the step that matters: the gap between the affordable model and the flagship narrowed to a hair — and where that lands for the work you actually run.

The lineup

The model most teams run all day.

Sonnet is the middle of three. It is where the everyday work lives.

HAIKU · fast, light
SONNET 5 · the everyday model, risen
OPUS 4.8 · the flagship

Sonnet sits between Haiku — fast and light — and Opus, the flagship. It is the default model on Free and Pro, and for most teams it handles the bulk of the real work. Sonnet 5 is the version where that everyday model closed most of the distance to the flagship.

The particulars. A full one-million-token context window at standard pricing, available in Claude Code and via the API as claude-sonnet-5 — the default for Free and Pro plans, and available to Max, Team, and Enterprise.

The honest read

Where Opus still wins.

On the toughest end-to-end coding, Opus 4.8 keeps a real lead — roughly six points. The read: for reasoning, analysis, writing, and the long agentic runs most work is made of, Sonnet 5 is now the value pick. For the deepest engineering problem in the building, the flagship still earns its price.

balance

Heads up · compare within the generation

Sonnet 5, like Opus 4.8, uses Anthropic’s newer tokenizer — it produces about 30% more tokens for the same text. That makes the Sonnet-5-to-Opus comparison apples-to-apples, but a per-word comparison against older Sonnet models tilts slightly in the older ones’ favor. The economics

Near-flagship work, at the mid-tier price.

The number that decides an agentic bill is output. Here it is, three ways.

The run is illustrative — twenty million output tokens on each lane, one dot per million. The rates are Anthropic’s published prices; flip the switch to see September’s rate. The ratio is what holds.

On the price that dominates a long agentic run — output tokens — Sonnet 5 runs $15 per million at standard, against Opus 4.8’s $25. That is 60% of the flagship’s rate for work that now lands close to it — and the same sticker the previous Sonnet carried. Through August 31, an introductory $10 puts it at 40%.

Why it matters

Autonomy you can afford to run.

The point is not a cheaper chat. It is a model that works on its own — at a price that lets you run it at scale.

“Claude Sonnet 5 is built to be the most agentic Sonnet model yet. It can make plans, use tools like browsers and terminals, and run autonomously at a level that, just a few months ago, required larger and more expensive models.” Anthropic · introducing Claude Sonnet 5

An agent that plans and works on its own spends far more than a single answer — it is the long autonomous run that costs. When near-flagship capability drops to the mid-tier rate, the arithmetic that kept those runs on the shelf changes. Work an executive team once queued for a person — reading a data export, drafting the first version of a brief, checking its own output — becomes something a machine can carry, at a cost that pencils out.
